# Workspace

A workspace is needed to start a new ML-project at isolated invoronment. So you can start each project and keep artifacts independently of others.  

A Workspase combines:
- compute resources, 
- object storage, 
- user permission,
- billing,
- priject settings. 

# Before creating a workspace, ensure:
- Billing account is created
- Payment method is attached
- Billing account status = **Active**
